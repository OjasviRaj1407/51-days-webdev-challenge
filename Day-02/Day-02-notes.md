# Web Development Day‑02

1) One page → One HTML heading (H1). 

2) `<hr />`, `<br />` → self‑closing tags.  

3) `<!-- QWERTY -->` → Comment tag.  

4) `li*5` → Creates 5 list items.  

5) Same level → treated as the same heading level.  

6) For a table → use the table tag: `<table>...</table>`. 

7) Use table rows to specify how many rows of data are inserted, i.e. number of rows → `<tr>...</tr>`.  

8) Use the table data tag to specify how much data needs to be inserted in a single row → `<td>...</td>`.  

9) Table header → `<th>...</th>`.  

10) `<thead>...</thead>`, `<tbody>...</tbody>`, `<tfoot>...</tfoot>` → used to specify the header, body, and footer of the table.
