### **WEB DEVELOPMENT DAY-02**



--> One page → One HTML heading (H1).



--> <hr />, <br /> → self‑closing tags.



--> <!-- QWERTY --> → Comment tag.



--> li\*5 → Creates 5 list items.



--> Same level → treated as the same heading level.



--> For a table → use the table tag: <table>...</table>.



--> Use table rows to specify how many rows of data are inserted, i.e. number of rows → <tr>...</tr>.



--> Use the table data tag to specify how much data needs to be inserted in a single row → <td>...</td>.



--> Table header → <th>...</th>.



--> <thead>...</thead>, <tbody>...</tbody>, <tfoot>...</tfoot> → used to specify the header, body, and footer of the table.
