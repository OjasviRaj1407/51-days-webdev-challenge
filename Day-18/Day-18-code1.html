Day 18 - CSS Box Model Practice

This page practices the CSS box model : content, padding, border, and margin using simple feature cards.

1. Feature Cards with Default Spacing

Each card is a box: margin (outside), border, padding (inside), and content.
Standard Card
padding : 16px

This card uses medium padding inside the border, so text has breathing room.

Tight Card
padding : 8px

Smaller padding makes the content feel more compact inside the same border.

Spacious Card
padding : 24px

More padding increases inner space, even though the border outline is similar.

2. Margin Between Elements

Margin controls the space outside the box, between one card and another.
Card Without Extra Margin

This card only has the row gap around it. No custom margin on the card itself.

Card With Extra Top Margin
margin-top : 12px

Extra top margin pushes this card down relative to the others, without changing its inner spacing.

3. Visualizing Content, Padding, Border, Margin

The divs below highlight how changing padding and border affects total visual size.
Box A - Base

padding : 16pxborder : 2px

This is a base box. Note how far the text is from the blue border.

Box B - More Padding
