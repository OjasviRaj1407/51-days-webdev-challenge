Day 17 – CSS Color Practice Page

This page is a quick reference and practice file for different CSS color formats : named colors, hex, rgb/rgba, and hsl/hsla.

1. Named Colors

These use predefined color keywords like teal, tomato, and slategray - easy to remember, limited in variety.

teal Used as a primary accent color.
tomato A strong call-to-action color.
slategray A muted background or text color.

2. Hex Colors (#RRGGBB)

Hex values like #1E88E5 or #FFB74D give precise control over color using the RGB model.

#1E88E5 Main brand blue (background, buttons).
#212121 Very dark gray for headers or footers.
#FFB74D Soft accent color for highlights.

3. RGB and RGBA

rgb() uses 0-255 values for red, green, and blue; rgba() adds an alpha channel (0-1) for transparency.

rgb(76, 175, 80) Solid green success color.
rgba(156, 39, 176, 0.7) Semi-transparent purple accent.
Overlay with rgba(0, 0, 0, 0.45) Transparent dark overlay on top of an image.

4. HSL and HSLA

hsl() uses Hue (0-360), Saturation (%), and Lightness (%). It’s great for generating consistent palettes by adjusting saturation and lightness.

hsl(210, 90%, 56%) Base brand blue.
hsl(210, 90%, 80%) Lighter version of the same hue.
hsl(210, 30%, 50%) More muted, desaturated blue.
hsla(0, 0%, 100%, 0.6) Glass-like white background for cards.
