# Web Development Day‑12

## 1. What is inline CSS?

1) Inline CSS means writing styles directly inside an HTML tag using the `style` attribute.  
2) It is applied to a **single element only**, so it is useful for quick demos, debugging, or one‑off changes, but not for large projects.  

## 2. Inline CSS basic syntax

1) Pattern: add `style="property: value;"` inside the opening tag.  
2) Example:  
   ```html
   <p style="color: blue; font-size: 18px;">This is styled with inline CSS.</p>
